# Student Task Manager

A full-stack Student Task Manager built using Next.js, Prisma, PostgreSQL (Supabase), Tailwind CSS, and Vercel.

## Features

- Add a task
- View all tasks
- Edit task title and description
- Update task status
- Delete tasks
- Search tasks

## Tech Stack

- Next.js
- React
- TypeScript
- Prisma ORM
- PostgreSQL (Supabase)
- Tailwind CSS
- Vercel
